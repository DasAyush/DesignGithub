Git and Github

GitHub is a web-based Git version control repository hosting service.

Is there a medium other than website to interact with this awesome site??

Answer: Technically No!!

Don't be sad dev's, there is another way to tackle this problem.
### GITHUB API * *_API URL_* : `https://api.github.com` * There are several authorization method available for use. 1. Basic Auth 2. OAuth (Sent in Header) 3. Oauth Token (Sent as parameter)
Retrofit

It turns your HTTP API into a Java interface.

Call method is associated with the Interface.

Call can perform Synchronous / Asynchronous call to request url.

Usage in Project

Parses JSON Responses fetched from the api. Serializes them into respective objects (Java). The data - structure is maintained using this api.
